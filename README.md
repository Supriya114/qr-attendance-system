# QR Attendance System

## 📌 Project Description
A Node.js + Express based QR Attendance System with login, signup, QR scanning, and manual attendance.

## 🚀 Features
- User Authentication (Signup/Login)
- QR Code Generation
- QR Code Scanning
- Manual Attendance
- Dashboard Management

## 🛠 Tech Stack
- Node.js
- Express.js
- HTML, CSS, JavaScript

## ▶️ Run Project
```bash
npm install
node server.js
